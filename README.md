## Overview
This project is a chatbot application built using Python, leveraging the power of OpenAI and FuzzyWuzzy for natural language processing. The project aims to provide a simple and interactive way to engage with users.

## Getting Started
To get started with the project, follow these steps:
1. Clone the repository: `git clone https://github.com/kalpana-ai/AI-group-3`
2. Install the required libraries: `pip install pandas openai fuzzywuzzy`
3. Set up your environment variables: create a `.env` file with your OpenAI API key

## Usage
To interact with the chatbot, simply run the `chatbot.py` script and follow the prompts.

## Configuration
* Set your OpenAI API key as an environment variable in the `.env` file
